# 📡 AI-радар — Новостной дайджест Клуба Незаменимых

## Что это

Автоматический ежедневный дайджест AI-новостей, публикуемый ботом в отдельный тред Telegram-группы клуба. Бот парсит RSS-ленты, фильтрует через LLM и выдаёт 3–5 самых значимых новостей за сутки — адаптированных под контекст клуба.

## Зачем

- **DAU:** участники заходят в клуб каждое утро за новостями — привычка, которая держит внимание.
- **Ценность:** качественная фильтрация экономит участникам 30–60 минут ежедневного скроллинга.
- **Маркетинг:** «мы всегда в тренде» — аргумент для удержания и привлечения.
- **Топливо:** новости питают обсуждения, инсайты и идеи для фокусов недели.
- **Стоимость:** близка к нулю при высоком эффекте. Работает полностью автоматически.

### Правила публикации

- **Время:** ежедневно в 09:00 MSK
- **Место:** отдельный тред `📡 AI-радар` в Telegram-группе клуба
- **Фоллбек:** если за сутки < 3 значимых новостей — пост не публикуется. На следующий день выходит расширенный дайджест за 48 часов. Лучше тишина, чем мусор.
- **Язык:** русский. Англоязычные заголовки переводятся.

## Алгоритм реализации (пошагово)

### Шаг 1 — RSS-парсер

Node.js скрипт. Библиотека: `rss-parser`. Запуск по крону в 08:30 MSK. Тянет 6 фидов, фильтрует по `pubDate` за последние 24 часа. Формирует JSON-массив:

```json
[
  {
    "title": "...",
    "description": "...",
    "link": "...",
    "source": "habr|vc|openai|huggingface|langchain|venturebeat",
    "pubDate": "2026-04-05T07:00:00Z"
  }
]
```

### Шаг 2 — AI-фильтрация

JSON отправляется в LLM (Claude API / OpenAI API) с системным промптом куратора (см. выше). На вход: 20–40 статей. На выход: готовый текст Telegram-поста с 3–5 отобранными новостями.

### Шаг 3 — Публикация

Готовый текст отправляется в Telegram через Bot API. Параметры: `chat_id`, `message_thread_id` (ID треда AI-радар), `parse_mode: HTML`.

### Шаг 4 — Мониторинг

Логировать в базу (Supabase):
- Дата
- Количество входных статей
- Количество отобранных
- Факт публикации (да/нет)
- Причина пропуска (если применимо)

Алерт фаундеру в личку, если бот не публикует 3 дня подряд.

## Расширения (после запуска MVP)

### Пятничный «Инсайт из радара»

Раз в неделю фаундер лично выбирает 1 новость из дайджестов за неделю и пишет комментарий: «Что это значит для незаменимых». Публикуется в `📣 Объявления`. Связывает автоматический бот с живым голосом фаундера.

### Реакции и голосование

Добавить к каждому посту inline-кнопки: `🔥 Полезно` / `💤 Мимо`. Собирать статистику реакций для калибровки промпта куратора.

### Тред разработки клуба

Отдельный тред для новостей о развитии продукта (из ТЗ):

| Топик | Назначение | Частота |
|---|---|---|
| **🛠 Разработка** | Роадмап, обновления, сбор фидбека | 1–2 раза/нед |

Контент: обновления продукта, опросы, роадмап, проделанная работа. Формат: ручной (не автоматический). Служит «щитом» от возражений «ничего не происходит» и инструментом сбора обратной связи.
