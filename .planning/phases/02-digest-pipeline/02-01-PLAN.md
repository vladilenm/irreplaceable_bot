Create `src/services/rss.service.ts` with:

1. Import RssParser from 'rss-parser' (default import), import FeedConfig, RawArticle from types, import logger.

2. Load feeds config: `const feeds: FeedConfig[] = JSON.parse(fs.readFileSync(new URL('../../config/feeds.json', import.meta.url), 'utf-8'));` — use fs.readFileSync with import.meta.url for ESM path resolution.

3. Create parser instance with a request timeout of 10 seconds:
```typescript
const parser = new RssParser({
  timeout: 10_000,
  headers: { 'User-Agent': 'NezamenimyeBot/1.0' },
});
```

4. Export async function `fetchFeeds(hoursBack: number = 24): Promise<RawArticle[]>`:
   - Calculate cutoff date: `new Date(Date.now() - hoursBack * 60 * 60 * 1000)`
   - Iterate over `feeds` array
   - For each feed, wrap in try/catch (per D-03: on error, log warning with feed name and error message, continue to next feed)
   - Parse feed with `parser.parseURL(feed.url)`
   - Filter items by `pubDate`: parse item.pubDate or item.isoDate, skip items without a valid date or with date before cutoff
   - Map valid items to RawArticle: `{ title: item.title ?? '', description: item.contentSnippet ?? item.content ?? '', link: item.link ?? '', source: feed.name, sourceKey: feed.sourceKey, pubDate: parsedDate }`
   - Collect all articles into a single array
   - Log info at end: `logger.info({ totalArticles: articles.length, feedsProcessed: successCount, feedsFailed: failCount }, 'RSS fetch complete')`
   - Return the combined array sorted by pubDate descending

Use ESM import style with `.js` extension for local imports. No `any` types — use rss-parser's built-in types or define minimal interfaces.
